# Live-weather-application
A basic weather application built with React and Vite. It fetches real-time weather data from the OpenWeatherMap API and displays temperature, humidity, wind speed, and weather conditions with custom icons. Perfect beginner project for learning API integration and dynamic UI updates.


---

# Weather App

This is a basic weather application built using React.
It uses the OpenWeatherMap API to fetch real-time weather data based on the city name entered by the user.

---

## Features

* Search weather by city
* Shows temperature, humidity, and wind speed
* Displays weather conditions using simple icons
* Clean and easy-to-use interface

---

## How to Run

1. Install dependencies:

   ```
   npm install
   ```
2. Add your OpenWeather API key in a `.env` file:

   ```
   VITE_WEATHER_API_KEY=your_api_key
   ```
3. Start the project:

   ```
   npm run dev
   ```
